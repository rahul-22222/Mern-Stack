-------------------------------Selectors
--------------
1. Selectors are used to select or target html elements to apply css
2.Types OPf Selectors 
        1.Simple  or Element Selector(Selects all tags of same type)
        2.Group Selector (Selects multiple tags of diffrent types in group)
        3.Class Selector(Selects Perticular tag with classname and tag name Combination)
